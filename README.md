| Page 3 | 🏆 Product Rankings |
| Page 4 | 👥 Customer Insights |
| Page 5 | 📦 Shipping Analysis |

---

## 📈 Key Visuals Built

- **Donut Chart** – Total Sales by Category (Accessories, Electronics, Gadgets)
- **Line Chart** – Total Sales by Year and Month (Jan 2023 – Nov 2023)
- **Bar Chart** – Total Sales by Product Name
- **Combo Chart** – Total Sales vs Total Profit comparison
- **KPI Cards** – Total Sales, Profit, Avg Order Value, Total Orders

## 🚀 How to Run

### Python Notebook
```bash
git clone https://github.com/Reyaz99578/E-commerce-data-analysis.git
cd E-commerce-data-analysis
pip install pandas matplotlib seaborn jupyter
jupyter notebook E_COMMERCE_SALES_ANALYST.ipynb




Power BI Dashboard
Open E_COMMERCE_PROFIT_DASHBOARD.pbix in Power BI Desktop
Click Refresh to load data
Use slicers to filter by Category / Date / Region
💡 Key Insights
📦 Gadgets dominate sales — contributing majority of 2.73M revenue
📅 Seasonal spikes visible in mid-2023 from line chart trends
💸 Profit margin ~21.6% indicates healthy business performance
🛒 High avg order value (3.28K) suggests premium product positioning
🏆 Top products: Cooling Pad & Bluetooth Speaker lead in sales volume
